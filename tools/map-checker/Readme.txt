===========================
=== Atrinik Map Checker ===
===========================

===========
= WINDOWS =
===========
To run Python scripts you need the latest Python 2.7.
Download installer: http://www.python.org/ftp/python/2.7.6/python-2.7.6.msi
Install it to the default folder.

For the map-checker specifically, you need PyGTK:
http://ftp.gnome.org/pub/GNOME/binaries/win32/pygtk/2.24/pygtk-all-in-one-2.24.2.win32-py2.7.msi
Install this to the default (Python) folder.

Use the start_gui.bat file to start the map-checker.
